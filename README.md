# M3U to Xtream Codes Generator – Firefox Extension

Official Website: https://xtreamiptv.de

M3U to Xtream Codes Generator is a lightweight Firefox extension that converts IPTV M3U playlist URLs into Xtream Codes login details instantly.

This tool extracts the server address, port, username, password, and generates the Xtream Codes API format for easy use with IPTV applications.

## Features

• Convert M3U links to Xtream Codes
• Fast and simple user interface
• Works directly inside the browser
• No external servers required
• No data collection or tracking

## Generated Information

The extension extracts the following details from the M3U URL:

* Server URL
* Streaming Port
* Username
* Password
* Xtream Codes API URL

## How to Use

1. Install the extension in Firefox.
2. Open the extension popup.
3. Paste your M3U playlist URL.
4. Click **Generate**.
5. Copy the generated Xtream Codes login details.

## Important Notice

This extension **does not provide IPTV channels, playlists, or streaming services**.
Users must provide their own legally obtained M3U playlist URL.

## Install Locally

1. Download this repository.
2. Open Firefox.
3. Navigate to **about:debugging**.
4. Click **Load Temporary Add-on**.
5. Select the `manifest.json` file.
